# Shared (referência do monorepo)

Tipos e contratos partilhados entre **web** e **mobile**.

A configuração Firebase em tempo de execução está espelhada em:

- `web/src/core/config/firebaseConstants.ts`
- `mobile/src/core/config/firebaseConstants.ts`

Mantenha estes ficheiros sincronizados ao alterar o projeto Firebase.
